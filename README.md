Calculartor

  A simple module which will do arithmetic operations..

Installation
  npm install pg-calc

Usage
  add:function(a,b){
    console.log(a+b);
  return a+b;
  },
  subtract:function(a,b){
    console.log(a-b);
  return a-b;
  },
  multiply:function(a,b){
    console.log(a*b);
  return a*b;
  },
  divide:function(a,b){
    console.log(a/b);
  return a/b;
  }

Tests
